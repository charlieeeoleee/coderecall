About Code Recall

Code Recall is a gamified learning system designed to help Computer System Servicing (CSS) students improve their understanding of technical subjects through interactive learning.

The system focuses on Computer Hardware and Electrical Wiring and Electronics Circuits, based on the results of a needs assessment conducted among students.

By integrating gamification elements such as points, levels, progress tracking, and interactive gameplay, Code Recall aims to increase student engagement and improve learning outcomes.
