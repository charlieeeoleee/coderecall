Code Recall is a gamified learning system designed for Computer System Servicing (CSS) students. It helps improve learning through interactive gameplay, quizzes, and progress tracking.

The system focuses on two main subjects: Computer Hardware and Electrical Wiring & Electronics Circuit, based on a needs assessment conducted among students.

Students earn experience points (XP), progress through levels, and receive feedback after answering questions. Interactive gameplay elements are used to enhance engagement.
