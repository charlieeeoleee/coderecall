Code Recall is a gamified learning platform designed to help Computer System Servicing students master computer hardware and electrical circuit concepts through interactive gameplay.

About Code Recall

Code Recall is a gamified learning system developed based on a needs assessment conducted among Computer System Servicing students. The system focuses on Computer Hardware and Electrical Wiring and Electronics Circuits to help students better understand complex concepts through interactive quizzes and gameplay.

Subjects Covered

🖥 Computer Hardware

Learn the internal components of a computer system, including motherboard, RAM, storage, and troubleshooting.

⚡ Electrical Wiring & Electronics Circuit

Understand electronic components, wiring systems, and basic circuit connections used in CSS.
